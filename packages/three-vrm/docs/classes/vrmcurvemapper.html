Class VRMCurveMapper

This is an equivalent of CurveMapper class defined in UniVRM. Will be used for VRMLookAtApplyers, to define behavior of LookAt.

See: https://github.com/vrm-c/UniVRM/blob/master/Assets/VRM/UniVRM/Scripts/LookAt/CurveMapper.cs
